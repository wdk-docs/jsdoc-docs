@copyright
=============================

.. rst:directive:: @copyright

   :Syntax: ``@copyright <some copyright text>``
   :Overview:

      The @copyright tag is used to document copyright information in a file
      overview comment. Use this tag in combination with the [@file tag]\ :rst:dir:`file`.

   :Examples:

      .. code-block:: js

         /**
          * @file This is my cool script.
          * @copyright Michael Mathews 2011
          */
